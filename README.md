# Excel---Word
Automatiza contratos con Excel y Word. Llena plantillas con datos de clientes, inserta tablas y guarda en PDF y Word. ✅ Valida archivos existentes para no sobrescribirlos y reporta el estado en tu Excel. Ideal para ahorrar horas en tareas repetitivas sin saber programar. Solo habilita macros y genera documentos al instante.
